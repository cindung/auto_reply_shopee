# auto_reply_shopee

Repo untuk project Auto Reply Shopee.

Konten singkat:
- Script utama: `A.js`
- Data sesi/peramban: folder `user-data-toko1/`, `user-data-toko2/`, `user-data-toko3/`
- Cookies: `cookies-toko1.json`, `cookies-toko2.json`, `cookies-toko3.json`

Cara upload dari komputer lokal (ringkas):
1. Buka terminal di folder project ini.
2. Inisialisasi git: `git init`.
3. Tambah semua file: `git add .`
4. Commit: `git commit -m "Initial commit"`.
5. Tambah remote: `git remote add origin https://github.com/cindung/auto_reply_shopee.git`.
6. Push: `git push -u origin main` (atau `master` sesuai cabang default).

Catatan otentikasi:
- GitHub sekarang biasanya memerlukan Personal Access Token (PAT) sebagai password, atau bisa menggunakan `gh auth login` (GitHub CLI).
- Jika push gagal, ikuti pesan error untuk melakukan login atau buat PAT.

Jika mau, saya bisa menjalankan langkah inisialisasi dan commit di komputermu, tapi untuk `push` kamu perlu memasukkan kredensial GitHub.
